QUICK START (2 steps)
======================
1) Drop your image into this folder and rename it to: shield.jpeg
2) Open index.html in your browser (or use VS Code Live Server).

TWEAKS (optional)
-----------------
- Change breathing speed: open styles.css and edit `--duration: 3s;` (lower = faster).
- Change glow intensity: tweak `--glow-min` and `--glow-max`.
- Change glow color: in @keyframes breathe, change the rgba() color.
